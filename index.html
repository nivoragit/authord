📌 Features

  • Documentations Tree View: Lists documentation instances defined in authord.json.
  • Topics Tree View: Displays topics corresponding to the selected documentation instance.
  • Dynamic Updates: Reflects changes in real-time in the topics directory.
  • Open Topics: Click on a topic to open its Markdown file in the editor.
  • Side-by-Side Preview: Open documents and view a live preview simultaneously for enhanced editing.

🚀 Getting Started

To use the Authord Docs Extension, follow these steps:

  1. Ensure you have authord.json in the root of your VS Code workspace.
  2. Create a topics directory containing Markdown documentation files.
  3. Launch VS Code and open the Authord sidebar to browse and open topics.

📝 Writerside Compatibility

Authord integrates with JetBrains' Writerside. You can open and continue Writerside projects, but new Writerside projects cannot be created within Authord.

⚠️ Known Issues

Currently, Authord Docs Extension supports only a single instance in authord.json. Future updates may introduce multi-instance support.
