Authord

Supercharge your VS Code documentation workflow with Authord: a modern, intuitive, and powerful extension designed to streamline your writing process.

FeaturesUsageWriterside CompatibilityKnown IssuesGitHub

Features

  • Documentations Tree View: Easily navigate your docs from authord.json.
  • Topics Tree View: Browse topics for each doc instance effortlessly.
  • Dynamic Updates: Real-time topic directory sync.
  • Open Topics: Instantly open Markdown files for edits.
  • Side-by-Side Preview: Live preview for faster writing.

Usage

Click the Authord icon in VS Code's Activity Bar to manage your documentation.

Writerside Compatibility

Continue JetBrains Writerside projects within Authord. You can't create new Writerside projects here, but you can seamlessly pick up existing ones.

Known Issues

Currently supports a single instance in authord.json. Multi-instance is planned.
